## Определение возраста покупателей

За основу взята свёрточная сеть ResNet50. За 3 эпохи модель достигла приемлемого значения метрики качества MAE. По итогу пяти эпох метрика достигла значения 6.3822, то есть созданная модель в среднем ошибается на 6 лет при определении возрата покупателя. Наиболее частые решения по разделению потребителей на возрастные группы предполагают переход из одной группы в другую через каждые 10-15 лет, соответственно обученная модель позволит достаточно точно подбирать специализированные предложения. Однако, для платежспособной группы людей от 18-25 лет ошибка все же существенная - интересы по покупкам в этом возрасте могут сильно отличаться в возрастной группе 18 и возрастной группе 25 лет. Поэтому, модель можно дорабатывать в дальнейшем. Этого же касается запрос на продажу алкоголя. При разнице в 6 лет алгоритм может ошибочно "позволить" продать алкоголь 14-15 летним подросткам, приняв их за 20-22 летних.
